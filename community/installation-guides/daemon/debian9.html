# Debian 9

In this guide we will install Pterodactyl's Daemon (wings) — including all of it's dependencies — and configure it to use a SSL connection.

  • Install Requirements
    • General Requirements
    • Docker
    • Nodejs
  • Installing the Daemon

TIP

This guide is based off the official installation documentation but is tailored specifically for Debian 9.

# Install Requirements

We will first begin by installing all of the Daemon's required dependencies.

# General Requirements

apt install -y zip unzip tar make gcc g++ python python-dev curl gnupg

# Docker

## install apt tools
apt install -y apt-transport-https ca-certificates curl gnupg2 software-properties-common

## Import the docker gpg key
curl -fsSL https://download.docker.com/linux/debian/gpg | apt-key add -

## Add the docker stable repo
add-apt-repository "deb [arch=amd64] https://download.docker.com/linux/debian $(lsb_release -cs) stable"

## Install docker
apt update -y
apt install -y docker-ce

systemctl enable docker
systemctl start docker

# Nodejs

## Install nodejs repo
curl -sL https://deb.nodesource.com/setup_10.x | bash -

## Install nodejs
apt install -y nodejs npm

# Installing the Daemon

Great, now all of the dependencies have been dealt with. From here follow the official Daemon installation documentation.
